# Community design notes

The package is being shaped around three adoption constraints:

1. **Cross-disciplinary readability** — non-method users should be able to understand the outputs.
2. **Workflow fit** — the package should work in notebooks, CLI jobs, pandas pipelines, neuro stacks, and agent loops.
3. **Communication-first artifacts** — summary cards, narrative reports, dataset cards, and compact contexts should travel easily between collaborators.

v0.9 especially emphasizes the third constraint by adding a docs-style homepage and an agent-driving layer.
