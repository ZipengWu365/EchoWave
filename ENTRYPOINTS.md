# Entry points

Use this file when you want the shortest path into the product:

- `start-here.html` — the single landing page for quickstart, live demo, environment doctor, and Pages export.
- `playground.html` — static showcase for reports and flagship demo cases.
- `tsontology-demo --open-browser` — local compute-backed demo server.
- `tsontology --guide doctor` — environment diagnostics for resolver noise and console encoding issues.
- `tsontology --export-pages docs` — create a Pages bundle ready for GitHub publishing.
